# CRUD-APPLICATION

This is a simple **CRUD APP** . 

✨Framework ✨-
 
 - node js
 - express js
 - mongodb
 - ejs
 - html
 - css
 - javascript
 
 ## Installation
 
 ```sh
npm install
```
## Then Create config.env file and create PORT and MONGO_URI Variable and specify Value. That's it. You are ready to go. To execute this project just type

 ```sh
npm start
```
